# claude-ai-agents

AI agents & browser-automation scripts built with Claude Code (TypeScript + Python).

- `agent.ts` — Composio-based agent
- `browser-agent.ts` — Agent SDK browser automation (reads `ANTHROPIC_API_KEY` from `.env`)
- `opera-gx-playwright.cjs` — Opera GX + Playwright automation
- `My_claude_script.py` — Anthropic API script

Copy `.env.example` → `.env` and add your keys (`.env` is gitignored, never committed).
